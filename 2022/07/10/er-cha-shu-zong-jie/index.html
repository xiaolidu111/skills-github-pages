二叉树的前中后序遍历

  • 前序遍历：就是刚进入一个节点的时候
  • 后续遍历：就是即将离开一个节点的时候
  • 中序遍历：就是左子树都遍历完，开始遍历右子树的时候做

两种解题思路

遍历一遍二叉树得到答案 分解二叉树

==104. 二叉树的最大深度==

  • 遍历一遍二叉树 

var maxDepth = function(root) {
    // 遍历一遍二叉树 用一个外部遍历记录二叉树的深度 前序位置更新二叉树的最大深度 然后遍历左右子树
    let depth = 0
    let maxDepth = 0
    function backTracking(root){
        if(root === null){
            return
        }
        depth++
        maxDepth = Math.max(depth,maxDepth) // 更新最大深度
        backTracking(root.left)
        backTracking(root.right)
        depth--
    }
    backTracking(root)
    return maxDepth
};
  • 分解子问题

var maxDepth = function(root) {
    // 分解子问题 先求左子树的最大深度 然后求右子树的最大深度 左后返回最大深度
    if(root === null){
        return 0
    }
    let left = maxDepth(root.left)
    let right = maxDepth(root.right)
    return Math.max(left,right) + 1
};

543. 二叉树的直径

因为要获取这个的最大的直径 以来左右子节点的深度 所以可以通过修改获取最大深度的代码来进行计算 


var diameterOfBinaryTree = function(root) {
    let maxDiameter = 0
    function getMaxDepth(root){
        if(root === null){
            return 0
        }
        let left = getMaxDepth(root.left) // 获得左节点的深度
        let right = getMaxDepth(root.right) // 右节点的深度
        maxDiameter = Math.max(left + right,maxDiameter) // 更新最大的直径
        return Math.max(left,right) + 1 // 返回左右节点的深度
    }
    getMaxDepth(root)
    return maxDiameter
};

层序遍历

主要思路

  • 利用一个队列,不断的入队出队来进行层序遍历 
// 输入一棵二叉树的根节点，层序遍历这棵二叉树
void levelTraverse(TreeNode root) {
    if (root == null) return;
    Queue<TreeNode> q = new LinkedList<>();
    q.offer(root);

    // 从上到下遍历二叉树的每一层
    while (!q.isEmpty()) {
        int sz = q.size();
        // 从左到右遍历每一层的每个节点
        for (int i = 0; i < sz; i++) {
            TreeNode cur = q.poll();
            // 将下一层节点放入队列
            if (cur.left != null) {
                q.offer(cur.left);
            }
            if (cur.right != null) {
                q.offer(cur.right);
            }
        }
    }
}

var invertTree = function(root) {
    if(root === null){
        return root // 如果为null 直接返回
    }
    let temp = root.left // 保存左节点

var connect = function(root) {
    if(root === null) return null // 如果为null 直接返回
    let queue = [] // 定义一个队列
    queue.push(root) // 放入到队列中
    while(queue.length){ // 当队列的长度有值
        let size = queue.length // 一层的长度
        for(let i = 0;i < size;i++){
            let node = queue.shift() // 逐个的弹出
            if(i !== size - 1){ // 让前弹出的等于下一个
                node.next = queue[0]
            }else{ // 如果是最后一个 让弹出的等于null
                node.next = null
            }
            node.left && queue.push(node.left) // 将不是空的节点防区队列
            node.right && queue.push(node.right)
        }
    }
    return root // 返回根节点
};

114. 二叉树展开为链表

  • 分解问题的思路

函数的定义 ,输入一个根节点 没有返回这 这个根节点就被拉成一个链表

主要流程

  • 先把左节点拉成一个链表
  • 再把右节点拉成一个链表
  • 左子树变成根节点的右子树 然后右子树拼到左子树上 

var flatten = function(root) {
    if(root === null) return root
    flatten(root.left)  // 左边的扁平化
    flatten(root.right) // 右边的扁平化
    let left = root.left // 获得左子树
    let right = root.right // 获得右子树
    root.left = null // 左子树值为空
    root.right = left // 右子树改为左子树
    let p = root // 找到此时右子树的最后一个
    while(p.right){
        p = p.right
    }
    p.right = right // 让右子树的最后一个等于这个原来的右子树
};

654. 最大二叉树

找到这个二叉树的最大值 生成一个根节点 然后递归调用这个函数 得到左子树和右子树 


var constructMaximumBinaryTree = function(nums) {  // 构造一个二叉树根节点 其中左右节点也构造完成
    function traversal(nums,left,right){ // 传入一个数组和范围  返回构造的二叉树
        if(left > right) return null // 如果左边大于右边就返回null
        let maxIndex = left // 初始化最大的位置
        let maxValue = nums[left] // 初始化最大值
        for(let i = left;i <= right;i++){
            if(nums[i] > maxValue){  // 遍历得到最大位置和最大值
                maxValue = nums[i]
                maxIndex = i
            }
        }
        let root = new TreeNode(maxValue,null,null) // 用最大值得到这个节点
        root.left = traversal(nums,left,maxIndex-1) // 调用函数得到这个的左节点
        root.right = traversal(nums,maxIndex+1,right) // 调用函数得到右节点
        return root // 返回根节点
    }
    return traversal(nums,0,nums.length-1)
};

105. 从前序与中序遍历序列构造二叉树

思路:

  • 先找到根节点的值,把根节点做出来,然后递归的构造左右子树
  • ==前序遍历的第一个节点就是根节点== ==找到前序遍历的根节点在中序遍历中的位置== ==然后得到这个根节点的左右子树在先序遍历和中序遍历的位置== ==最后递归调用计算结果== 

var buildTree = function(preorder, inorder) {
    if(preorder.length === 0) return null // 如果现需遍历的数组长度是0 就直接返回
    let val = preorder.shift() // 得到根节点的值
    let index = inorder.indexOf(val) // 得到根节点的坐标
    let root = new TreeNode(val,null,null) // 生成根节点
    root.left = buildTree(preorder.slice(0,index),inorder.slice(0,index))// 递归构造 先序遍历就是0到index 中旬遍历也是0,index
    root.right = buildTree(preorder.slice(index),inorder.slice(index+1)) // 现需遍历就是index到最后,中序遍历是index+1栋最后
    return root // 返回根节点
};

106. 从中序与后序遍历序列构造二叉树

思路
由中序遍历和后续遍历的特点可知,后续遍历的根节点在最后一个前序遍历的根节点在第一个
==所以大致过程和前一题相似== 


var buildTree = function(inorder, postorder) {
    if(inorder.length === 0) return null // 判断没有元素的情况
    let val = postorder[postorder.length - 1] // 找到根节点
    let index = inorder.indexOf(val) // 找到对应的下标
    let root = new TreeNode(val,null,null) // 生成根节点
    root.left = buildTree(inorder.slice(0,index),postorder.slice(0,index)) // 生成左子树 其实是0到index  中序遍历是0,index
    root.right = buildTree(inorder.slice(index+1),postorder.slice(index,postorder.length-1)) // 中序是index+1到最后  后续是index到最后一个但是不包括最后一个
    return root // 返回根节点
};

889. 根据前序和后序遍历构造二叉树

思路:
这题和前面两题有明显的区别 通过前序和中续,后续和中序可以得到一个唯一二叉树 但是前序和后续无法确定一个唯一的二叉树
具体过程:

  • 首先吧前序遍历和后序遍历的第一个和最后一个元素作为根节点的值
  • 吧前序遍历结果的第二个元素作为左子树的根节点的值
  • 在后序遍历中寻找左子树根节点的值,从而确定了左子树的索引边界 进而确定右子树的索引边界 递归的构造左右子树即可 

function getLongestSequence (input) {
  // write code here
  let start = 0
  let end = 0
  let resCount = 0
  let resChar = input[0]
  while (end < input.length) {
    let str = input[start] // 记录当前的字符
    if (str !== input[++end]) { // 如果不相同
      let count = end - start// 出现的次数
      if (count > resCount) {
        resCount = count
        resChar = str
      }
      start = end // 开始和结束在相同的位置
    }
  }
  let result = ''
  for (let i = 0; i < resCount; i++) {
    result += resChar
  }
  return result
}

297. 二叉树的序列化与反序列化

就是将这个转换成字符串 然后还可以将字符串转换成二叉树

那么这个可以转换为 2,1，#，6，#，#，3，#，# 然后可以通过这个转换成二叉树

其实就是考察二叉树的遍历方式，下面讲解三种方式

前序遍历解法

对于这个序列化函数 可以用前序遍历 将其放入一个数组 里面当遇到这个null的时候 就放入#来占位
具体代码 


var serialize = function (root) {
  let list = [] // 生成一个数组用来储存这个字符串的每个字符
  function mySerialize (root) { // 传入一个根节点
    if (root === null) {
      list.push('#') // 如果是空就放入# 然后返回
      return
    }
    list.push(root.val) // 如果不是空就放入这个的值
    mySerialize(root.left)
    mySerialize(root.right) // 递归处理左右子树
  }
  mySerialize(root)
  return list.join(',') // 拼接成字符串
};

在前文的时候,我们使用这个前中后序遍历的结果需要两个互相配合才能还原二叉树,那是因为前文的遍历结果没有激励空指针的信息.这里的node列表包含了空指针的信息,所以只使用node列表就可以还原二叉树
反序列化的过程也是大致相同的,先确定根节点,然后遵循前序遍历的规则,递归生成左右子树即可
就是第一个元素是根节点 只要吧第一个元素作为根节点 后面交给递归函数取解决就行了 

var deserialize = function (data) {
  let nodes = data.split(',') // 先把字符串转换成这个节点
  function myDeserialize (nodes) {
    // 接受一个节点列表
    if (nodes.length === 0) {
      return null // 如果这个没有了就返回null
    }
    let first = nodes.shift() // 第一个就是 头结点
    if (first === '#') return null // 如果第一个是#那就返回null
    let root = new TreeNode(first) // 生成根节点
    root.left = myDeserialize(nodes)
    root.right = myDeserialize(nodes) // 递归处理左右的节点
    return root
  }
  return myDeserialize(nodes) // 借用辅助函数来完成这个转化
};

中序遍历解法

中序遍历行不通 因为反序列化行不通,按照之前的思路 反序列化需要知道根节点 但是中序遍历 不能知道根节点 所以行不通

层序遍历解法

通过对层序遍历的框架进行修改 让这个层序遍历能够记录null指针 


var serialize = function (root) {
  // 通过层序遍历完成序列化
  let res = [] // 序列化数组
  let queue = [] // 生成一个队列
  if (root == null) return '#' // 如果是null直接返回#
  queue.push(root) // 否则吧根节点放入这个队列中
  while (queue.length) { // 当队列不为空
    let node = queue.shift() // 弹出头部
    
    if (node === null){// 如果这个是空 就不进行下一步了
      res.push('#')
      continue
    } 
    res.push(node.val)
    queue.push(node.left)
    queue.push(node.right)
  }
  return res.join(',') // 最后返回拼接的字符串
};

每一个这个根节点都对应这两个子节点 所以反序列化也是从这个角度出发 


var deserialize = function (data) {
  let nodes = data.split(',') // 分隔成数组
  if (nodes.length === 0) return null // 如果没有 就返回这个null
  // 第一个元素就是根节点的值
  let rootVal = nodes[0]
  let root = new TreeNode(rootVal) // 生成根节点
  let queue = [] // 初始化队列
  queue.push(root) // 放入根节点
  for (let i = 1; i < nodes.length;) {
    // 遍历这个nodes数组
    let parent = queue.shift() // 弹出父节点
    let left = nodes[i++] // 左节点就是下一个
    if (left !== '#') {
      // 如果下一个不是空 就让这个等于父节点的左子树
      parent.left = new TreeNode(left)
      queue.push(parent.left) // 把左子树压入到这个队列中
    } else {
      parent.left = null  // 否则这个左子树就是null 不需要进入队列
    }
    let right = nodes[i++] // 这个是右子树
    if (right === '#') {
      // 道理和左子树一样
      parent.right = null
    } else {
      parent.right = new TreeNode(right)
      queue.push(parent.right)
    }
  }
  return root
};

652. 寻找重复的子树

思路
计算以一个节点为根的子树是否有重复子树,需要的信息

  • 以自己为根的二叉树长什么样子
  • 以其他节点为根的子树长什么样子

要拿到以自己为根的子树 可以使用后序遍历 然后通过拼接字符串的方式将这个二叉树序列化 用#表示空指针 用,表示间隔
然后借助一个外部的数据结构 把每个子树的序列化结果都存进去 这样对于每个节点 就可以知道是不是有重复的了 

var findDuplicateSubtrees = function (root) {
  // 在外部定义一个储存序列化结果的哈希表
  let hash = {}
  // 先定义一个序列化函数
  let res = [] // 储存结果
  function traverse (root) {
    if (root === null) {
      return '#'
    }
    let left = traverse(root.left)
    let right = traverse(root.right) // 对左右序列化
    let subTree = left + ',' + right + ',' + root.val // 生成序列化结果
    hash[subTree] = (hash[subTree] || 0) + 1 // 记录出现的个数
    if (hash[subTree] === 2) { // 出现两次 加入到这个结果数组中
      res.push(root) // 将根节点加入到这个res中
    }
    return subTree
  }
  traverse(root)
  return res
};

归并排序详解及其应用

所有的递归算法,本质上都是在遍历一棵树,然后在节点(前中后序)的位置上执行代码,你要写递归算法,本质上就是告诉每个节点需要做什么
归并排序就是先把左半部分数组排好序,再把右边数组排好序,然后把两半数组合并
归并排序就是用到了分解问题的思路 


/**
 * @param {number[]} nums
 * @return {number[]}
 */
 var sortArray = function(nums) {
// 归并排序需要两个函数 一个是将数组分成两部分 一个是将两部分的数组合并
  if(nums.length === 1) return nums // 如果数组的长度是1 就直接返回这个数组就可以了
  let mid = Math.floor(nums.length / 2) // 取出中间的位置 为了将数组分成两部分
  let left = nums.slice(0,mid) // 左边的部分
  let right = nums.slice(mid) // 右边的部分
  return merge(sortArray(left),sortArray(right)) // 递归的调用sort函数进行排序得到两个数组  然后将两个数组进行merge
};
function merge(left,right){
  // merge函数 参数是两个数组 一个是左 一个是右
  let m = left && left.length // m 是左边数组的长度
  let n = right && right.length // n 是右边数组的长度
  let result = [] // 结果数组
  let i = 0 // 遍历左边数组的下标
  let j = 0 // 遍历右边数组的下标
  while(i < m && j < n){
    if(left[i] < right[j]){
      // 如果左边数组的值小于右边数组的值
      result.push(left[i++]) // 将左边数组的值放入结果数组中
    }else{
      // 否则将右边结果的值放入结果数组中
      result.push(right[j++])
    }
  }
  // 退出循环 判断左右数组是不是有生意
  while(i < m){
    result.push(left[i++])
  }
  while(j < n){
    result.push(right[j++])
  }
  return result // 返回合并后的数组
}

二叉搜索树中的删除操作 450. 删除二叉搜索树中的节点

总体思路:先找再改

对于找到之后 也要分情况

  • 没有子节点 那就直接删除就好了
  • 有一个子节点 让子节点顶替自己的位置 然后删除原来的节点
  • 有两个子节点
    • 找到左子树的最大值 替换当前的节点 这是第一个方案
    • 找到右子树的最小值替换当前节点 这是第二个方案 
var deleteNode = function(root, key) {
    if(root === null) return root // 如果根节点是null 直接返回root
    if(root.val === key){ // 如果找的值就是当前的值 那就删除这个节点
        if(root.left === null){ // 判断左子树是不是null 如果是 就返回右子树就行了
            return root.right 
        }else if(root.right === null){ // 判断右子树是不是null 如果是那就返回左子树就行了
            return root.left
        }else{ // 如果都不是null 那就要删除这个节点了
            let cur = root.right // 向右子树 找到最小的节点
            while(cur.left){ // 小的节点都在左边
                cur = cur.left
            }
            cur.left = root.left // 这个时候cur就是最小的节点了 然后将这个原来的根节点的左子树 放到cur的左子树上 这样才满足二叉搜索树的性质
            return root.right // 返回删除后的右边的子树
        }
    }
    if(root.val < key) root.right = deleteNode(root.right,key) // 如果找的值大于根节点的值 那就去右边节点去删除
    if(root.val > key) root.left = deleteNode(root.left,key) // 如果找到的值大于根节点的值 那就去左边节点删除
    return root
};

236. 二叉树的最近公共祖先

思路:
两个节点的最近公共祖先其实就是这两个节点向根节点的延长线的交汇点,那么对于任意一个节点,怎么知道自己是不是pq的公共祖先被
如果一个节点能够早左右子树中找到pq,则该节点为LCA节点
那么只需要在原来的后续遍历的基础上修改一下代码就能找到最近的公共祖先的解法了
在后续的位置,如果发现左右节点的位置都是非空那么这个节点就是公共祖先了 


var lowestCommonAncestor = function(root, p, q) {
    function find(root, p, q){
        if(root === null){
            return null
        }
        if(root.val === p.val || root.val === q.val){ // 注意的是节点 需要对比值
            return root
        }
        let left = find(root.left,p,q)
        let right = find(root.right,p,q) // 向左右去寻找
        if(left != null && right != null){
            // 如果左右都不是这个null 说明这个就是最近的公共祖先
            return root
        }
        return left != null ? left : right
    }
    return find(root,p,q)
};

    }

235. 二叉搜索树的最近公共祖先

思路在之前的思路中,是判断这个节点是否存在来判断是不是这个节点,在二叉搜索树中可以利用到二叉搜索树的性质
首先要分析出来这两个的最大最小值
如果当前节点的值大于最大值 那就直接去左子树中寻找就行了
如果当前节点的值小于最小值 那就直接去右子树中寻找就行了
否则的话 当前节点的值在这两个的中间 那就直接返回就行了 


var lowestCommonAncestor = function(root, p, q) {
    let max = Math.max(p.val,q.val)
    let min = Math.min(p.val,q.val)
    function find(root,max,min){
        if(root === null){
            return root // 如果是null 直接返回
        }
        if(root.val > max){
            return find(root.left,max,min)
        }
        if(root.val < min){
            return find(root.right,max,min)
        }
        return root // 在中间直接返回
    }
    return find(root,max,min)
};

1650. 二叉树的最近公共祖先 III

var lowestCommonAncestor = function(p, q) {
    let a = p
    let b = q // 缓存p和q节点
    while(a !== b){ // 寻找相交的链表
        if(a){
            a = a.parent
        }else{
            a = q
        }
        if(b){
            b = b.parent
        }else{
            b = p
        }
    }
    return a
};

完全二叉树的含义
每一层都是紧凑靠左排列的,除了最后一层每层都是满的

开始解题:怎么求一个树的节点个数呢
对于普通的二叉树,遍历一遍节点就行了
对于满二叉树:节点的个数和层数是指数的关系 


var countNodes = function(root) {
    let h = 0
    while(root !== null){
        h++
        root = root.left
    } // 退出的时候root等于null
    // h就是层数
    return Math.pow(2,h) - 1
};

对于完全二叉树,并不能仅仅的按照满二叉树来进行计算,而是要考虑左边和右边的层数,具体代码见下 

var countNodes = function(root) {
    let left = root // 遍历左边节点
    let right = root // 遍历右边节点
    let hl = 0
    let hr = 0 // 左右的层数
    while(left !== null){
        hl++
        left = left.left
    }
    while(right !== null){
        hr++
        right = right.right
    }
    if(hl === hr){
        // 如果两个相同 相当于满二叉树
        return Math.pow(2,hl) - 1
    }
    return 1 + countNodes(root.left) +countNodes(root.right) // 否则返回左子树的个数+右子树的个数 因为一直划分下去 肯定两边都是满二叉树
};
